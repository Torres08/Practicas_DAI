Práctica 6:   Single Page Application con Vite y React


12, 13 de Diciembre
José María Guirao (jmguirao@ugr.es)

En esta práctica haremos una SPA que muestre un escaparate de la tienda como este. Usaremos la librería de UI React, las clases de css Tailwind, y alguna librería de componentes sobre Tailwind como daisyUI u otra similar.

Los datos los traemos del api fakestoreapi, con la librería swr.

Primero instalamos la herramienta de desarrollo, los componentes css y el hook de react para el api.

Para renderizar listas en react, hay que añadir un atributo key distinto para cada elemento de la lista, para que react distinga un elemento de otro a la hora de actualizarlo.


Para nota

  • Poner a funcionar el botón Buy now de cada producto, para que muestre una página de detalle del producto (como en Sharing data between components).
